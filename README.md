# 📄 Resume Builder – Backend API

A scalable and secure Resume Builder backend built using Spring Boot.
This project provides REST APIs for user authentication, resume creation, profile management, PDF generation, and cloud-based file storage.

---

# ✨ Features

* 🔐 User Authentication & Authorization (JWT Based)
* 📝 Create, Update & Delete Resumes
* 📂 Multiple Resume Sections

  * 🎓 Education
  * 💡 Skills
  * 💼 Experience
  * 🚀 Projects
  * 📜 Certifications
* 📧 Email Based Notifications
* 📄 Resume PDF Generation
* ☁️ Cloud Image/File Upload Support
* 🔗 Secure REST APIs
* 🛡️ Role-Based Access Control
* 🗄️ Database Integration
* ⚠️ Exception Handling & Validation
